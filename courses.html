CEN4213: Embedded Systems Programming

Embedded systems programming is a more valuable discipline than ever, driven by fast-growing, new fields such as wearable technology. This course is an introduction to the skills needed to write the efficient embedded code necessary to make tomorrow’s devices work. It focuses on building a system with all the key components hardware developers must deliver to manufacturing.

CIS4517: Needs Assessment & Tech Transfer

This course is designed to provide you with skills to analyze and solve business computing needs so that you can showcase what you have learned in your other courses. This is a capstone course and will complete project in a group collaborative enviroment.

COP4365: Advanced C# Programming

This course is designed to provide the student with an understanding of .NET and C# programming. The course covers the following topics: the C# language; .NET IDE and tools, including debugger; net architecture; web services; web applications; windows applications; mobile device applications; and interoperability of using different types of applications.
